# Tableau-Dashboard-H R Analytics

Built interactive dashboard to track and analyze employee attrition for 1470 employees
Used parameters, filters, and slicers to drill down by age, salary, job role, education, and tenure
Connected tables and wrote calculations to compute attrition rate, avg tenure, and key HR metrics
Created custom visuals like donut, bar, line charts, and tables to highlight high-risk employee segments
Skills : Calculation, Data Modeling, Data Visualization, HR Analytics, Parameters, Filters.
